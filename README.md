Vistas implementadas (simulando celular a traves de una modal) 
1) Vista Principal (cuadricula de fotos y lista de carpetas)
- barra de busquedad (head)
- galeria de fotos (body)
- boton flotante en el footer (solo visible en vista cuadricula)
2) Vista de imagen 
- Nombre de la imagen 
- botones inferiores (sin funcionalidad)
Funcionalidades simples implementadas: 
1) Cambiar la vista de fotos (cuadricula) a folders (lista) con el boton en la barra de busquedad (tal cual como en la app original).
Para esto solo agrupo segun la carpeta que indica la ruta de la imagen
------------------------------------------------------------
Para ejecutar la pagina y se aprecie correctamente: 

1) Clonar el proyecto
2) Abrirlo en vs Code 
3) Instalar una extension para visualizar en tiempo real (Live Server)
4) Ubicarse en el archivo .html
5) click derecho y buscar la opcion "open with live server" 
6) Listo, se deberia abrir su navegador predeterminado mostrando el contenido del html y  las imagenes adjuntas en sus carpetas
